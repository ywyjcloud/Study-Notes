
# 写作说明

1. 例子都用`###` 开头
2. 例子解法用`####` 开头
3. 需要强调的内容用 `####` 开头, 或者`**`加粗
    1. 如果使用`####`, 下面必须有说明,否则只用加粗;
    2. 连续两行都需要强调的内容,只能用`**`加粗
    3. 能加粗,就不用`####`
4. 
## 查看Vim内置文档

1. `:h XXXX`
> 比如`:h vimtutor`查看vim向导

## 本文档的编码说明

1. `Ctrl-s`,`<C-s>` 表示同时按住`Ctrl`键和`s`键
2. `d`:按一次`d`键
3. `daw`:依次按`d`，`a`，`w`键

## 使用Vim出厂设置

`vim -u NONE -N`
