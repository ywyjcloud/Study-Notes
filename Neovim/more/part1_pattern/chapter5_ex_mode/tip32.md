# 技巧32： 自动补全Ex命令
> `<Tab>`自动补全

1. `<C-d>` 会列出所有可用的补全列表
2. `<Tab>` 会依次补全为候选（按一次，环一个候选）

#### `wildmenu` 选项

1. 在`vimrc`中设置`set wildmenu`后，vim会提供补全导航列表
> 1. `<Tab>`, `<C-n>`, `<Right>` 正向遍历列表
> 2. `<S-Tab>`, `<C-p>`, `<Left>` 反向遍历列表
