# 技巧31： 重复上次的Ex命令

1. `:@:` 重复上次的Ex命令
2. `:@@` 使用过`@@`之后再重复之前的指令，使用`：@@`
3. `:bn` 等价于 `:bnext` 在缓冲区列表中正向移动
4. `:bp` 等价于 `:bprevius` 在缓冲区列表中逆向移动
5. `<C-o>` :每次使用`:bnext` 的时候都会在跳转列表中添加一条记录，`<C-o>` 会回到跳转列表的上一条记录处 (理解为**回看**)
