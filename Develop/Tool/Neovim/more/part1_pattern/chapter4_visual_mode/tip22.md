# 技巧22: 重复执行面向行的可视命令
> `.`命令对于面向行的可视命令可复用

### 例子: 调整python代码的缩进

![tip22](../../images/tip22.png)  

1. `Vj` 选中两行
2. `>` 将选中的行向右缩进一层
3. `.` 重复缩进操作`>`

> **Note** <br>
> 1. `>.` 也可以用 `2>`替换,但是不如`>.`的及时视觉反馈来得快
