# 技巧 104: 不用离开 Vim 也能编译代码
> vim 提供了快速跳转到出错位置的方法

1. `make`命令在shell中编译
2. `:make`在vim中调用编译
> 和系统的`make`输出一样，但是vim会做一些分析，列出其中的错误信息。对于错误信息，会在 quickfix 列表中为其创建一项记录。
> 1. 使用`:cnext`跳转到quickfix列表的下一处出错位置
