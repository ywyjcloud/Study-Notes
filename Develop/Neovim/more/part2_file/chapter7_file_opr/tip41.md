# 技巧41： 用`:edit`命令打开文件

> 1. `:edit`命令允许通过绝对路径和相对路径打开文件

shell 命令的`pwd`显示当前的路径（`print working directory`)

## 相对于活动文件目录打开一个文件

1. `:edit %<Tab>`
> `%`表示当前工作路径，`<Tab>`将其展开

2. `:edit %:h<Tab>`
> `%:h` 表示当前工作区文件所在文件夹的完整路径

**note** <br>
> 在vimrc中添加`cnoremap <expr> $$ getcmdtype() == ':' ? expand('%:h').'/':'%%'`，则每次在命令行模式中输入`%%`，都会自动显示当前工作目录路径
