# 技巧25: 修改列文本

### 例子: 替换CSS中的图片路径

> 将 `/images/` 替换成 `/components/`

![tip25](../../images/tip25.png)  


1. `<C-v>jje`: 进入列可视模式(`<C-v>`), 选中三行(`jj`), 每列选中到单词结尾(`e`)
2. `c`: 删除选中文本
3. `<Esc>`: 显示剩下两行的修改结果
