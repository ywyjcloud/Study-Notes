# 技巧24: 用面向__列块__的可视模式编辑**表格数据**

### 例子: 将表格的每一列之间用竖线隔开(每行添加竖线)

![tip24](../../images/tip24.png)  

1. `<C-v>3j` 选中四列,每列宽1
2. `x..` 每行删3列
3. `gv` 重选上次的高亮区域
4. `r|` 将光标所在字符替换成 `|`
5. `Vr-` 选中光标所在行(`V`), 将光标所在的字符替换成`-`
