# 动词

## vt

> **T**ransitive Verb	-	及物动词
>
> 
>
> 后接**宾语**(动作作用的对象)	She is ==writing== code.





## vi

> **I**ntransitive Verb	-	不及物动词
>
> 
>
> 动作完整	She is ==laughing==.
